======================================
ИНСТРУКЦИЯ ПО ЗАПУСКУ ЧАТ-БОТА
Terra Danza Bot
======================================

1. УСТАНОВКА PYTHON
-------------------
Если Python не установлен:
1. Скачайте Python 3.10+ с https://www.python.org/downloads/
2. При установке отметьте галочку "Add Python to PATH"
3. Проверьте установку: python --version

2. СОЗДАНИЕ БОТА В TELEGRAM
---------------------------
1. Откройте Telegram и найдите @BotFather
2. Отправьте команду /newbot
3. Введите имя бота (например: Terra Danza Bot)
4. Введите username бота (должен оканчиваться на _bot)
5. Скопируйте полученный токен вида: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

3. ПОЛУЧЕНИЕ СВОЕГО ID
----------------------
1. В Telegram найдите @userinfobot
2. Отправьте /start
3. Скопируйте ваш числовой ID (например: 123456789)

4. НАСТРОЙКА КОНФИГУРАЦИИ
-------------------------
Откройте файл config.py и замените:
- YOUR_BOT_TOKEN_HERE на токен от BotFather
- 123456789 на ваш ID

Пример:
BOT_TOKEN = "7123456789:ABCdefGHIjklMNOpqrsTUVwxyz"
ADMIN_ID = 712345678

5. УСТАНОВКА ЗАВИСИМОСТЕЙ
-------------------------
Откройте командную строку в папке bot и выполните:

pip install -r requirements.txt

6. ЗАПУСК БОТА
--------------
В командной строке выполните:

python main.py

При успешном запуске вы увидите:
"Запуск бота..."
"База данных инициализирована"

7. ТЕСТИРОВАНИЕ
---------------
1. Найдите вашего бота в Telegram по username
2. Отправьте /start
3. Протестируйте все функции:
   - О проекте (карта танцев, история, курсы)
   - FAQ (5 вопросов о Terra Danza)
   - Оставить заявку (опрос из 5 вопросов)
   - Связаться с командой проекта

8. ФАЙЛЫ ПРОЕКТА
----------------
bot/
├── main.py           - Главный файл бота
├── config.py         - Конфигурация (токен, FAQ, вопросы)
├── database.py       - Работа с SQLite
├── requirements.txt  - Зависимости
├── bot.log           - Логи (создаётся при запуске)
└── bot_database.db   - База данных (создаётся при запуске)

9. ПРОСМОТР ДАННЫХ
------------------
База данных SQLite хранится в файле bot_database.db

Просмотреть можно:
- Через DB Browser for SQLite (https://sqlitebrowser.org/)
- Через Python:
  python -c "import sqlite3; c=sqlite3.connect('bot_database.db'); print([r for r in c.execute('SELECT * FROM leads')])"

10. ОСТАНОВКА БОТА
------------------
Нажмите Ctrl+C в командной строке

======================================
О ПРОЕКТЕ TERRA DANZA
======================================
Terra Danza — интерактивное образовательно-культурное 
веб-приложение, объединяющее историю мирового танца 
в единую цифровую систему.

Миссия: решать проблему разрозненности и недостоверности 
информации о народных и исторических танцах, создавая 
структурированную платформу с географической привязкой 
и проверенными данными.
